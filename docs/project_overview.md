# Project Overview

`PD-and-scorecard-commercial` is the main commercial borrower-rating and scorecard engine in the public credit-risk stack.

## Portfolio role

It converts borrower financial analysis and industry overlays into PD estimates, score bands, and policy-style decision outputs.

## Upstream inputs

- `financial-statement-analysis`
- `industry-analysis`

## Downstream consumers

- `LGD-commercial`
- `expected-loss-engine-commercial`
- `stress-testing-commercial`
- `RAROC-pricing-and-return-hurdle`
- `RWA-capital-commercial`
